# Pizza Sales Data Analysis Project

## Project Overview
This project analyzes pizza sales data to uncover key insights regarding customer preferences, sales trends, and revenue generation.

## Key Insights Tracked
- **Total Revenue & Total Orders**: Tracking overall business performance.
- **Monthly Revenue Trends**: Analyzing sales peaks and drops across different months.
- **Pizza Category & Size Analysis**: Understanding which types and sizes of pizzas are most popular among customers.
- **Top & Worst Selling Pizzas**: Identifying the best-performing and least-performing items by revenue and quantity.

## Tech Stack Used
- **SQL Workbench**: For data manipulation and running key business queries.
- **Power BI Desktop**: For building interactive visualizations and dashboards.
